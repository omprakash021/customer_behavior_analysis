# Customer Behavior Analysis Project

## Overview

This project demonstrates an end-to-end Data Analytics workflow, from data preparation to business insights. The objective is to analyze customer purchasing behavior, identify trends, and create interactive dashboards that support data-driven decision-making.

The project covers data loading, exploratory data analysis (EDA), data cleaning, SQL analysis using PostgreSQL, dashboard development in Power BI, report creation, and presentation design.

---

## Dataset

The dataset contains customer purchase information, including:

* Customer ID
* Age
* Gender
* Location
* Purchase Amount
* Product Category
* Item Purchased
* Payment Method
* Review Rating
* Subscription Status
* Customer Segment

## Project Steps

### 1. Data Loading

* Imported the dataset using Python.
* Verified data structure and column information.

### 2. Exploratory Data Analysis (EDA)

* Analyzed distributions and trends.
* Identified missing values and outliers.
* Generated visualizations for key variables.

### 3. Data Cleaning

* Removed duplicates.
* Handled missing values. (37 Review Rating was missing)
* Standardized data formats.
* Prepared data for analysis and reporting.

### 4. PostgreSQL Database Integration

* Connected Python to PostgreSQL.
* Created database tables.
* Loaded cleaned data into PostgreSQL.

### 5. SQL Analysis

Performed business-focused SQL queries such as:

* Customer segmentation analysis
* Top-selling product categories
* Revenue analysis
* Purchase behavior trends
* Product popularity analysis

### 6. Power BI Dashboard

Created an interactive dashboard to visualize:

* Customer demographics
* Product performance
* Purchase trends
* Revenue insights
* Customer segmentation

### 7. Report Creation

Prepared a detailed analytical report summarizing:

* Key findings
* Business insights
* Recommendations

### 8. Presentation Development

Created a professional project presentation using Gamma to communicate results effectively.

## How to Run

### 1. Clone the Repository

git clone <repository-link>

### 2. Install Required Libraries

pip install pandas numpy matplotlib seaborn sqlalchemy psycopg2-binary

### 3. Run Python Analysis

Open the Jupyter Notebook and execute all cells.

### 4. Load Data into PostgreSQL

Configure PostgreSQL credentials and run the database loading script.

### 5. Execute SQL Queries

Run the SQL scripts provided in the project to generate analytical insights.

### 6. Open Power BI Dashboard

Open the `.pbix` file and refresh the data connection.

### 7. Review Report and Presentation

Explore the project report and presentation files for a summary of findings and recommendations.
